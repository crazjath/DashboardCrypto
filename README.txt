Instalaciones

Para poder ejecutar deuna correcta manera el archivo es necesario contar con NodeJs y MongoDB instalados,
esto por el uso de paqueterias de NodeJs y MongoDB sera nuestra base de datos.

Paqueterias a instalar:

Express:(##)

$npm install Express

handlebars:(##)

$npm install handlebars

mongodb:(#Erto para poder hacer uso de la base de datos#)

$npm install mongodb

nodemon:(##)

$npm install nodemon

tailwindcss:(##)

$npm install tailwindcss


Uso:
Para poder hacer uso del server por parte de nodemon se ejecutara la siguiente linea en la
terminla:

$npm run dev

Al momento de hacer uso de las herramientas de tailwindcss se ejecutara un script para que
se pueda crear el diseno en el archivo css, por lo que al momento de hacer cambios en css
se ejecutara el siguiente script:

$npm run build-css

Esto es obligatorio correr aunque sea un cambi minimo, es necesario para que se pueda 
crear el css
